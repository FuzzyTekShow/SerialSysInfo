Reading the Serial Data

The data sent from the application to the serial device is in the following format:

CPUTEMP|GPUTEMP|CPUCOREFREQUENCY|GPUCOREFREQUENCY|GPUMEMFREQUENCY|CPULOADPERCENTAGE|GPULOADPERCENTAGE|USERAM|TOTALRAM|USEDRAMPERCENTAGE|CURRENTIME|CURRENTDAYCURRENTMONTH|

Here is an example of real world serial data colour coded for reading convenience:

56.75|43|3680.018|1252.573|3610.272|8.333333|11.36697|6.777683|15.92783|42.55247|16:58|Mon27|

CPU Temp|GPU Temp|CPU Freq|GPU Core|GPU Mem Freq|CPU Load Percentage|GPU Load Percentage|Used RAM|Total RAM|RAM Percentage|Time (00:00)|Day & Date (DAY00)|

The vertical pipe ("|") character is used to indicate the end of a part of data.
The received data can then be manipulated in any way you wish.
For example the CPU Frequency of "3680.018" from above could be converted from Mhz to Ghz, the CPU temperature could be displayed to only 1 decimal and so on.
