The GUI

The GUI contains all the needed controls in one simple to use window.

In the screenshot above, the elements numbered are as follows:
  1. This is where you select the serial port to connect to
  2. The baud rate for the data transfer. Default is 9600
  3. How often should the data be updated and sent to the serial device (In seconds)
  4. Whether or not the data in the window (see 8) will also be updated
  5. If ticked, the software will start when you login to Windows.
  6. When the software is started, it will be minimized to the tray right away
  7. Upon launch, send serial data straight away without having to press the button (see 9)
  8. The GUI area containing the metric data
  9. Start or stop the sending of serial data
  10. Saves and applies any changed you might have made to the settings area
  11. Minimizes the software to the system tray
  12. Exits the program
